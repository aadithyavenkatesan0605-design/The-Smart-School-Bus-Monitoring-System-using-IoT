# The-Smart-School-Bus-Monitoring-System-using-IoT

The Smart School Bus Monitoring System enhances student safety during school transportation by integrating IoT technology and FaceNet-based facial recognition. Traditional systems like manual attendance and basic GPS tracking cannot ensure that only authorized students board the bus or provide complete monitoring.

This system uses facial recognition to authenticate students, allowing the servo-controlled bus door to unlock only for registered students while automatically marking attendance. The system also tracks the bus location using GPS and updates all information in real time on a cloud platform, allowing parents and school administrators to monitor the journey.

Additionally, an emergency panic button sends instant alerts to the cloud application during critical situations. By combining facial recognition, GPS tracking, and IoT-based monitoring, the system provides a secure and efficient solution for school bus transportation.
